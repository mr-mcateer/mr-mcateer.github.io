Toolbox Certification Day

Mr. McAteer -- Automotive Manufacturing -- Crescent Valley HS
Teacher Agenda -- Do Not Distribute

Timeline

0:00 - 0:05
Briefing
Hand out audit forms. Explain objective: every toolbox gets inventoried, organized, and certified today. Point out the QR code for the Tool Identifier. Assign toolbox colors to engine groups.
0:05 - 0:10
Empty and sort
Students empty toolboxes completely onto benches. Group by type. Seniors begin teacher cabinet audit.
0:10 - 0:30
Audit
Students work through the 20-tool checklist. Rate condition, mark action codes. Pull extras to the Orphan Bin. This is the main work block -- circulate and check progress.
0:30 - 0:40
Mystery tools and extras
Students ID unknowns using the reference sheet, phone Tool Identifier, and seniors. Log extras. Seniors float to help.
0:40 - 0:50
Clean and organize
Wipe tools, arrange drawers (top = sockets, middle = wrenches, bottom = everything else). Bonus: matching set hunt.
0:50 - 0:55
Certification spot-checks
Each group brings their form. Spot-check 3 items per box. Sign off if accurate. Flag boxes that need attention.
0:55 - end
Wrap-up
Collect all audit forms. Review Orphan Bin -- redistribute or set aside for ordering. Note any critical shortages.

Materials Needed

  • Printed audit forms (1 per toolbox = 6 copies)
  • Printed master tool list (1 per toolbox = 6 copies, laminate after)
  • Orphan Bin on the front bench (empty box or tray)
  • Rags / shop towels for wiping tools
  • Pen per group (students fill in forms by hand)
  • Phone access for students (Tool Identifier via QR code)

After Class

  • Review all 6 audit forms -- compile missing tool list for ordering
  • Redistribute tools from Orphan Bin to understocked boxes
  • Laminate master tool lists and tape inside each toolbox lid
  • Route strays: metals shop, vehicle area, instructor cabinet, shared shelf
  • Note any boxes rated CRITICAL for priority restocking
